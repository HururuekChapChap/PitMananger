# PitMananger
 3-2학기 소프트웨어공학 수업 경매채팅프로그램
# 개발기간
 약 2개월
# 개발인원
 1명
# 개발언어
 java
# 프로그램 설명
 교수님 요구사항 : 1:N 채팅, 친구추가, 친구상태확인, 로그인, 로그아웃
 
 차별화 기능: 경매방 생성 및 삭제, 방입장, 호가, 1:1 채팅
